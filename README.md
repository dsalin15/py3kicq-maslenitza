<h1 align="center">py3kicq</h1>
Библиотека для работы с протоколом OSCAR (используемом в ICQ) на языке Python 3.

## Установка

1. Скачайте последнюю версию Python с [официального сайта](https://python.org/).
2. Скачайте весь репозиторий в отдельную папку.
3. Получите два UIN на [официальном сайте KICQ](http://kicq.ru/reg117623/) (или на другом сервере, где вы хотите воспользоваться данной библиотекой).
4. Вставьте уин и пароль в соответствующие строки.
5. Запустите, используя эту команду
```
python3 maslenitza.py
```
## ВАЖНО! При отключении бота, пожалуйста, вводите программу !logout ! Бот будет принимать комманду только от вашего админ уина, который вы указали в соответствующей строке.
